## 🛠️ 构建方法

```bash
# 克隆仓库
git clone https://github.com/NyaNyagulugulu/ba.git

# 构建（默认为1.21.1版本）
./gradlew build

# 构建其他版本，例如1.16.5
./gradlew build -PmcVersion=1_16_5
```
